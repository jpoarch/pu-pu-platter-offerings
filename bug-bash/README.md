# Bug Bash

Given a list of issues and access to the code repository.  The community will "squash" as many bugs as quickly as possible.  This will include ~85 bugs

## Input

- Source code access
- Build scripts

## Best Practice Assumptions
- Existing Test cases

## Exclusions

## Deliverables

- Test case
